# Homebrew Tap for Dotsquare Labs

This is the official [Homebrew](https://brew.sh) tap for [Dotsquare Labs](https://github.com/Dotsquare-Labs) projects.

## Install

```bash
brew tap Dotsquare-Labs/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `nudge` | Get a text when your script finishes. CLI companion for [nudge.sh](https://nudge.sh) |

### nudge

```bash
brew install nudge
```

## How It Works

Formulae in this tap are automatically updated by [GoReleaser](https://goreleaser.com) when new versions are released in their respective repositories.
